# alfred-workflow

## Prerequisites

- **OS**: macOS — the bundled scripts use macOS-only tools (`plutil`, `zip`, `osascript`).
- **Alfred 5** with the **Powerpack** — required to install and run the `.alfredworkflow`
  files this skill produces (workflows are a Powerpack feature).
- **Programs**: `python3` (runs `scripts/validate.py`). `plutil` and `zip` ship with macOS.
